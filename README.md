# Joel Nave Portfolio

Static HTML site for Joel Nave Portfolio.

Requirements:

  * [Node.js](http://nodejs.org/).

To set-up your environment execute:

    $ npm install

To build execute:

    $ npm run build

To test:

    $ npm run start

And open http://localhost:8000 in your browser.

To clean build:

    $ npm run clean
